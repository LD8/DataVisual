## installation command:

* [matplotlib](https://matplotlib.org/gallery/) 

  ```bash
  $ python3 -m pip install --user matplotlib
  ```

  

* [plotly](https://plot.ly/python/)

  ```bash
  $ python -m pip install --user plotly
  ```
